B.8 Shutdown Log Messages in Telnet Shell

When you use the shutdown shell command to stop Virgo Tomcat Server, the shutdown log messages appear in the shell terminal instead of in the terminal in which Virgo runs. See Enabling Equinox Console for more information. This is covered also by bug 351533.

7.2 Starting in Clean Mode

When you start Virgo Server for Apache Tomcat in clean mode, the startup script removes the SERVER_HOME/work directory (and hence all running applications) as well as all trace, log and dump files. It leaves the SERVER_HOME/repository and SERVER_HOME/pickup directories untouched, which means that any applications previously hot deployed will be automatically reinstalled.

Linux

To start Virgo Server for Apache Tomcat in clean mode, open a terminal window and run startup.sh -clean: 

prompt$ cd $SERVER_HOME
prompt$ bin/startup.sh -clean

Microsoft Windows

To start Virgo Server for Apache Tomcat in clean mode, open a command window and run startup.bat -clean: 

prompt> cd %SERVER_HOME%
prompt> bin\startup.bat -clean

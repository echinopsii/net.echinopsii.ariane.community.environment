B.5 Problem Deleting Installation Directory under Windows

Sometimes Microsoft Windows won't let you delete the Virgo Server installation directory, typically because of long paths inside the work directory.

You can return the Virgo instance to a clean state by stopping Virgo if necessary and then running the startup script with the options -clean -noStart, after which you should be able to delete the installation directory. See Cleaning Virgo Server for Apache Tomcat without Starting it for more information.
